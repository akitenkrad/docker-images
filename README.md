# Quick reference
- docker environments for python

# Supported tags
- python.cpu:latest
- python.gpu:latest

# Python Environment
- miniconda: python 3.9

# How to use

## for cpu environment

```
docker run --name python -d akitenkrad/python.cpu:latest
```

## for gpu environment
```
docker run --name python -d akitenkrad/python.gpu:latest
```
